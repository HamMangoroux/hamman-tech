Meaning

This post is supposed to be the actual planning webpage. I will be updating this page and maybe adding more details I realized needed more time or whatever. I’ll split it into phases so that the stuff is clear. I won’t predict how long I’ll finish but I will write down when I start that phase and when I end up finishing.

Phase 1 - Reading & Initial Setup

Plan Update:

2026-05-25: This phase is just reading the LFS guide and getting a good idea. This phase will be pretty slow and it’s just to give me an idea of how things will be like. I also can’t do too much because my school’s finals are approaching so I gotta LOCK IN on those. I’ll just be reading the guides on the bus or something (Lies, I’ll be sleeping on the bus).

Complete:

  1. Finish reading the LFS guide.
  2. Look over relevent areas of BLFS (way too long and a lot of it is KDE or GNOME).
  3. Ask around the discord and other communities for common issues
  4. Finish eSSD partition setup and get environment ready.

Phase 2 - Simple Linux CLI Setup (Sysvinit)

2026-05-25: In this phase, I’ll just be following the LFS and complete my build of a very minimal linux setup. This setup will be using Sysvinit although in later phases, we will change that over to OpenRC. I’ll add more content to this once I start with the date stamp.

Complete:

  1. Use the LFS guide to configure simple CLI Linux base setup.

Phase 3 - Simple BLFS

2026-05-25: So in this one we just add a couple base apps along with SwayWM. Again, I really don’t have a good idea of what I’ll be doing in this step so yeah… I’ll be adding to this section though.

Complete:

  1. Make Simple GUI
  2. Simple tools (i.e. gef, nvim, ghidra)

Phase 4 - RICE FIELDS

2026-05-25: In this step, I’ll be making it look all pretty and stuff. A lot of the configs are already in my other setups so I’ll just copy it over and maybe using CSS to do some custom stuff. I’ll try not to go overboard. This is supposed to be a simple distro.

Complete:

  1. Riced applications and desktop environment
  2. Custom keybinds
